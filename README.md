## üöÄ Getting Started

### üì¶ Prerequisites

- Node.js
- MongoDB
- Firebase account for authentication

### Setup Instructions

#### Clone the repository:

  ```
      git clone https://github.com/tanishkadeep/Regal-Estate.git
      cd regal-estate
  ```

#### Backend Setup

1. Navigate to the backend directory & install dependencies:

   ```bash
   cd api
   npm install
   ```

2. Create a .env file:

   ```
   MONGO=your_mongodb_connection_url
   JWT_SECRET_KEY=your_jwt_secret_key
   ```

3. Start the backend server:

   ```
   npm run dev
   ```

#### Frontend Setup

1. Navigate to the frontend directory & install dependencies:

   ```
   cd client
   npm install
   ```

2. Create a .env file:

   ```
   VITE_FIREBASE_API_KEY=your_firebase_api_key
   ```

3. Start the frontend server:

   ```
   npm run dev
   ```
